objects

We are learning a language.
Ni lernas lingvon.

I am eating an apple.
Mi manĝas pomon.

You love the brother.
Vi amas la fraton.

subject - verb - object.

subject - does the action.
object - receives the action.  the action is done on it.

there is no preposition in front of an object.
also called the direct object.

in English, there is no marker on the object.  word order tells us which is the object.

in Esperanto, the -n suffix is added to a noun when it is the object of the verb.

Notice:  the earlier set of verbs do not take an object - try it!  it does not work.

havi - to have
ami - to love
fari - to do or make

try leaving the object off - generally it does not sound right.

Li amas la virinon.
Ŝi manĝas pomon.
Ni faras la laboron.

Adjective agrees with Noun

Li amas la belan virinon.
Ŝi manĝas la ruĝan pomon.
Ni faras la facilan laboron.

in Esperanto, the -n suffix is added to an adjective when it describes the object of the verb.

kiu and tiu (correlatives)

kiu /noun/ - which /noun/
tiu /noun/ - that /noun/

kiu hundo - which dog
kiu frato - which brother

tiu viro - that man
tiu arbo - that tree

kiu - which one (or who, if you know it's a person)
tiu - that one

kiu estas vi?
kiu estas la viro?

tiu estas Miĉele.
tiu estas la avo.
